# 警察向け逮捕状管理ポータル（デモ／教育用）

このリポジトリは、**教育目的・フロントエンド学習用**の逮捕状管理ポータルのサンプルです。  
実際の運用や機密情報管理には使用できません。

---

## 概要

- HTML/CSS/JavaScript で構築した管理画面のデモ
- ログイン認証はフロントエンド側のダミーで、誰でもログイン可能
- 逮捕状データは架空のデータを使用
- CSVエクスポートやモーダル表示などの UI を学習できる

---

## 目的

- フロントエンドでのテーブル管理・検索・フィルタ・モーダル表示の実装例を学ぶ
- 教育用のサンプルとして、ログ・監査画面の簡易デモも含む

---

## 注意事項

- **実運用では絶対に使用しないこと**  
- 認証・認可、TLS/暗号化、監査ログ、入力検証などは全てサーバー側で実装する必要があります
- 個人情報や機密情報は含まれていません
- GitHub などに公開しても安全ですが、実際の逮捕状管理システムではありません

---

## デモ機能

1. フロントエンドでのログイン（ダミー）
2. 逮捕状一覧の表示・検索・フィルタ
3. モーダルで逮捕状詳細表示
4. CSV エクスポート（ダミー）
5. 新規逮捕状作成（フロントエンドのみ）

---

## 使用方法

1. このリポジトリをクローン
2. `index.html` をブラウザで開く
3. 職員IDは任意で入力可能（パスワードもダミー）
4. デモ機能を試すことができます

---

## ライセンス

このリポジトリのコードは、**学習・教育目的のみ**に使用してください。
